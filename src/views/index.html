API Endpoints

Todos

  • Create (/todos) POST

    body:
    • title

  • index(/todos) GET

    queries:
    • sort=asc/dec
    • limit(integer)
    • completed=true/false

  • Show(/todos/:id) GET

  • Create (/todos/:id) PATCH

    body(must contain at least one of):
    • title
    • completed

  • Delete(/todos/:id) DELETE
